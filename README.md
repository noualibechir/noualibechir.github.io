# Bechir Nouali — Portfolio
Single-page portfolio. Link-only (noindex). Browser tab icon = your photo.

############################################################
#  DEPLOY ON GITHUB  (~5 min, free)
############################################################
1) github.com → sign up (free).
2) "+" (top-right) → New repository.
   • Name EXACTLY:  yourusername.github.io   (e.g. noualibechir.github.io)
   • Public → Create repository.
3) Add file → Upload files → drag in: index.html, README.md, the whole assets/ folder → Commit changes.
4) Settings → Pages → Branch: main, Folder: / (root) → Save.
5) ~2 min later → live at  https://yourusername.github.io
Update later: re-upload the changed file → Commit.

############################################################
